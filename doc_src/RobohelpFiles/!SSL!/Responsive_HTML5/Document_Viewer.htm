Document Viewer

The Document Viewer opens in a new web browser window when you click an encrypted file listed under your listed repository. You can use the following options to view your NXL protected document in the Document Viewer:

 

Feature

 Description

 

 

Click to navigate to the first page.

 

 

Click to navigate to the last page.

 

 

Click to navigate to the previous page.

 

 

Click to navigate to the next page.

 

 

Click to fit your document vertically in the document viewer

 

 

Click to fit your document horizontally in the document viewer

 

 

Click to rotate your document anticlockwise.

 

 

Click to rotate your document clockwise.

 

 

 

Click to zoom in to your document.

 

NOTE: The pinch zoom gesture is disabled for NXL protected documents that are viewed on a mobile device. You can instead use this icon to zoom in on your screen.

 

 

 

 

Click to zoom out of your document.

 

NOTE: The pinch zoom gesture is disabled for NXL protected documents that are viewed on a mobile device. You can instead use this icon to zoom out on your screen.

 

 

 

 

Click to print your document.

 

NOTE: You can only print a document if you have sufficient permission. This icon is not visible if you are viewing your document using a mobile web browser.

 

 

If you are viewing a VDS file, then you can use the following options:

 

Feature

 Description

 

 

Click to toggle the display of each steps's description.

 

 

Click to return to the first step.

 

 

Click to play the current step.

 

 

Click to pause at the current step

 

 

Click to play all steps from step1 onwards.

 

 

 

Click to print the current view of your VDS file.

 

NOTE: You can only print if you have sufficient permission. This icon is not visible if you are viewing your document using a mobile web browser.

 

 

If you are viewing a CAD file, you can use the following options:

 

Feature

 Description

 

 

 

Click to measure the distance between two points.

 

Click on any area of your model and click a second time to mark the second point to which you want to measure.

 

NOTE: Each measurement you make on your model is represented in the Model Browser window, under Measurements. You can select each measurement to view it on your model and also toggle it from view.

 

 

 

 

Click to measure the angle between two points.

 

NOTE: Each measurement you make on your model is represented in the Model Browser window, under Measurements. You can select each measurement to view it on your model and also toggle it from view. This option does not work for all types of models.

 

 

 

 

Click to select a part.

 

The part is highlighted in green color and is also selected in the Model Browser window.

 

NOTE: Select a part in the Model Browser window and it is highlighted in the display area.

 

 

 

Click to explode all parts.

 

 

Click to trim your view based on the x, y, or z axis.

 

Click to print the current view of your file.

 

NOTE: You can only print if you have sufficient permission. This icon is not visible if you are viewing your document using a mobile web browser.

 

 

Model Browser

The Model Browser is displayed in the Document Viewer when you are viewing JRT or PRT files. It contains a list of all the parts that are displayed in the Document Viewer. You can toggle each part and subpart of your model from view.

 

The Model Browser also lists all measurement annotations (the distance between two points and the angle between two points) under a Measurements section. These annotations can also be toggled from view.

Properties window

The Properties window displays when you select a particular part in your model. It displays the Name of the part and the part's Parent.

 

In the Properties window, you can use the following options for the selected part:

Option

 Description

 

Parent

 

 

Click to select the parent of the selected part.

 

 

Transparency

 

 

Click to view the transparent version of the selected part.

 

 

Bounding Box

 

 

Click to view the bounding box dimensions of the selected part.

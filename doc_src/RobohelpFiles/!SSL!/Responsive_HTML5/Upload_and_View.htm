Upload & View

Rights Management Server is a web portal to upload and view NextLabs rights protected documents in a secure manner. Click here for the list of supported formats. RMS interacts with the Policy Controller before it displays any rights protected content for a user.

 

If due to a policy you are not authorized to view the document you have uploaded, Rights Management Server displays a message indicating this using a generic message.

 

This message is displayed for all occurrences of any policy denying you the ability to view the file. Any specific obligation messages that are part of the policy (which prevents you from viewing the uploaded file) are not displayed.

 

  1. Login to the Rights Management Server portal.
  2. Click Upload.
  3. Click Browse and select your rights protected document.
  4. Click Open.

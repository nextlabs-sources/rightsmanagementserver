Manage Service Providers

This page displays three sections:

 

  • Configured Providers - Displays a list of available service providers that are integrated with Rights Management Server (RMS). This allows  users to add repositories for the given service providers in RMS.
  • Available Providers - Displays a list of available service providers that RMS supports. You can configure these service providers to integrate with RMS to grant users access to the repository from RMS. After the service providers are configured, it will move to the Configured Providers. Currently, RMS supports Google Drive, OneDrive, SharePoint Online, SharePoint On Premise, and Dropbox service providers.
  • Cross-Launch Apps - Displays a list of service providers that can be integrated with RMS  to allow users view the rights protected or unprotected files directly from these service providers. This feature is supported only by SharePoint On Premise and SharePoint Online apps.

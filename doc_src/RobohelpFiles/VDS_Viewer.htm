VDS Viewer

The VDS Viewer opens in a new web browser window when you click on a rights protected or unprotected VDS files listed under your repository. You can use the following options to view your document in the VDS Viewer.

 

Feature

 Description

 

 

Click to toggle the display of each steps's description.

 

 

Click to return to the first step.

 

 

Click to play the current step.

 

 

Click to pause at the current step.

 

 

Click to play all steps from step one onwards.

 

 

 

Click to print the current view of your VDS file.

 

NOTE: You can only print if you have sufficient permission. This icon is not visible if you are viewing your document using a mobile web browser.

 

Click to download a rights protected file.

Click to view the file type, size, classification labels, and file rights such as print, view, etc.

 

Click the arrow to view the next scene.

Click the arrow to view the previous scene.

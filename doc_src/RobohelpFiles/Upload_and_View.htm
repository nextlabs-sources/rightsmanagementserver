Upload & View

Rights Management Server allows users to upload and view NextLabs rights protected documents and unprotected documents in a secure manner. Click here for the list of supported formats. If you are not authorized to view the document you have uploaded, Rights Management Server displays a message indicating that you are not authorized to view the file.

 

Perform the following steps to view a document:

 

  1. Login to the Rights Management Server portal.
  2. Click Upload.
  3. Click Browse and select your rights protected document.
  4. Click Open.

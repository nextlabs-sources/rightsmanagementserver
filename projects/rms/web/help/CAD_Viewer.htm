Model Browser

The Model Browser is displayed in the CAD Viewer when you are viewing the CAD files except the dwg files. It contains a list of all the parts that are displayed in the CAD Viewer. You can toggle each part and subpart of your model from view.

 

The Model Browser also lists all measurement annotations (the distance between two points and the angle between two points) under the Measurements section. These annotations can also be toggled from view.

Properties window

The Properties window displays when you select a particular part in your model. It displays the Name of the part and the part's Parent.

 

In the Properties window, you can use the following options for the selected part:

 

Option

 Description
